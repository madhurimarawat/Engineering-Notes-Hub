## 📑 Index – List of Experiments

### 1️⃣ Setting Up a Virtual Machine in a Cloud Environment

Learn the basics of cloud computing by creating and configuring a virtual machine using platforms like AWS, Google Cloud, Azure, or LocalStack.

### 2️⃣ Deploying a Web Application on a Cloud Server

Deploy a simple web application on a cloud server (e.g., AWS EC2) to understand cloud-based hosting.

### 3️⃣ Working with Cloud Storage Services

Create and manage an AWS S3 bucket to upload, verify, and download data securely.

### 4️⃣ Setting Up and Configuring Cloud Networking

Configure networking components such as VPCs and subnets to understand cloud network architecture.

### 5️⃣ Using Cloud Functions for Serverless Computing

Deploy and test AWS Lambda functions using AWS CLI and LocalStack to simulate serverless execution.

### 6️⃣ Cloud Load Balancing and Auto Scaling

Demonstrate load balancing and auto-scaling concepts using a Flask application to simulate traffic handling.

### 7️⃣ Cloud Databases and Data Management

Deploy and manage a PostgreSQL database using Docker to simulate a scalable cloud database setup.

### 8️⃣ Cloud Security: Identity and Access Management (IAM)

Configure users, roles, and policies using AWS IAM to implement secure access control mechanisms.

### 9️⃣ Implementing Cloud Monitoring and Logging

Set up AWS CloudWatch to monitor resources, configure alarms, and analyze logs for system reliability.

### 🔟 Setting Up a Cloud-based CI/CD Pipeline

Implement a CI/CD pipeline using GitHub Actions to automate testing, building, and deployment processes.
